5.1 推荐教材与读物

5.1.1 Stata 编程基础

  • Baum, C. F. (2016). An Introduction to Modern Econometrics Using Stata. Stata Press.

  • Cameron, A. C., & Trivedi, P. K. (2010). Microeconometrics Using Stata. Stata Press.

  • Gould, W. W., Pitblado, J., & Poi, B. (2018). Stata User’s Guide. Stata Press.

5.1.2 可复现研究与数据科学

  • Gentzkow, M., & Shapiro, J. M. (2014). Code and data for the social sciences: A practitioner’s guide. University of Chicago.

  • Wilson, G., et al. (2017). Good enough practices in scientific computing. PLOS Computational Biology, 13(6).

  • Peng, R. D. (2011). Reproducible research in computational science. Science, 334(6060), 1226-1227.

5.1.3 计量经济学方法

  • Angrist, J. D., & Pischke, J.-S. (2008). Mostly Harmless Econometrics: An Empiricist’s Companion. Princeton University Press.

  • Wooldridge, J. M. (2010). Econometric Analysis of Cross Section and Panel Data. MIT Press.

  • Cunningham, S. (2021). Causal Inference: The Mixtape. Yale University Press.

5.1.4 AI 与数据分析

  • Molnar, C. (2022). Interpretable Machine Learning. Lulu.

  • Marcus, G., & Davis, E. (2019). Rebooting AI: Building Artificial Intelligence We Can Trust. Pantheon.

5.5.2 推荐扩展命令

* 数据处理
ssc install ftools, replace   // 快速分组计算
ssc install gtools, replace   // 高效数据操作
ssc install tidy, replace     // 数据整理

* 估计方法
ssc install reghdfe, replace  // 高维固定效应
ssc install did_multiplegt, replace  // 多期DID
ssc install csdid, replace    // Callaway-Sant'Anna DID
ssc install drdid, replace    // 双重稳健DID

* 输出与可视化
ssc install estout, replace   // 表格输出
ssc install coefplot, replace // 系数图
ssc install grstyle, replace  // 图形样式
ssc install palettes, replace // 颜色调色板

* 工具类
ssc install ietoolkit, replace // 影响评估工具包
ssc install texdoc, replace    // LaTeX文档生成

5.6 进一步学习路径

5.6.1 初级 → 中级

  1. 掌握本课程内容
  2. 阅读 Wooldridge (2010)
  3. 完成一个完整的实证项目
  4. 学习 Git 版本控制

5.6.2 中级 → 高级

  1. 学习机器学习在计量中的应用
  2. 深入因果推断方法
  3. 学习 Python/R 作为补充工具
  4. 参与开源项目贡献

5.6.3 高级 → 专家

  1. 研究前沿计量方法
  2. 开发 Stata 外部命令
  3. 在学术期刊发表论文
  4. 分享知识、指导他人
